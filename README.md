**Spanish**
Sistema para Gestion de Reservas de un Restaurante

Este es un proyecto inicial para un sistema de gestión de reservas para un restaurante.
El backend será desarrollado en Java con Spring Boot y el frontend será desarrollado con Android Studio.

**English**
Restaurant Reservation Management System

This is an initial project for a reservation management system for a restaurant. The backend will be developed in Java with Spring Boot and the frontend will be developed with Android Studio
